Các bước cần làm:

- Dùng npm:

1. Chạy câu lệnh: npm install --omit=dev

2. Để chạy project, tại root: npm start

=============

- Dùng yarn:

1. chạy câu lệnh: yarn
2. chạy project: yarn start

=============

- Dùng bun:

1. chạy câu lệnh: bun
2. chạy project: bun start

=============
